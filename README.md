# react-typescript-project-template

## 环境要求

nodejs v8.0 +

## 本地开发

```bash

# 安装依赖
yarn

# 启动
yarn start

# 打包
yarn build

# 代码风格检查

# 访问
访问(自动打开) http://localhost:8080/
```

## 相关技术栈

- react@16+
- webpack@4+
- react-hooks
- babel@7
- typescript@3.7.0
- react-router-dom@4
- react-router@4
- antd@3
