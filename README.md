# react-typescript-project-template

## 环境要求

nodejs v8.0 +

## 本地开发

```bash

# 安装依赖
yarn

# 启动
yarn start

# 打包
yarn build

# 打包分析
yarn build:analyzer

# 代码风格检查
yarn lint

# 访问
访问(自动打开) http://localhost:8080/
```

## 相关技术栈

- react
- webpack
- react-hooks
- babel
- typescript
- react-router-dom
- antd
- es-build
